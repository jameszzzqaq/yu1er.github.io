121. 买卖股票的最佳时机#

遍历过程中维护一个股票价格最小值和一个股票价格最大差值.

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int floor = prices[0];
        int ans = 0;
        for (int i = 1; i < prices.size(); i++) {
            ans = max(ans, prices[i] - floor);
            floor = min(floor, prices[i]);
        }
        return ans;
    }
};
  • 时间复杂度: $O(n)$
  • 空间复杂度: $O(1)$

122. 买卖股票的最佳时机 II#

  1. 贪心
class Solution {
public:
    int maxProfit(vector<int>& prices) {

        int ans = 0;
        for (int i = 1; i < prices.size(); i++) {
            if (prices[i] > prices[i-1]){ 
                ans += prices[i] - prices[i-1];
            }
        }
        return ans;
    }
};
  • 时间复杂度: $O(n)$
  • 空间复杂度: $O(1)$
  1. 动态规划
  • buy表示i时刻买入股票的最大收益
  • sell表示i时刻卖出股票的最大收益

$$ \left{ \begin{aligned} buy = max(buy, sell - prices[i]) \
sell = max(sell, buy + prices[i]) \
\end{aligned} \right. $$

最大值为$sell$

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int n = prices.size();
        int buy = -prices[0], sell = 0;
        for (int i = 1; i < n; i++) {
            buy = max(buy, sell - prices[i]);
            sell = max(sell, buy + prices[i]);
        }
        return sell;
    }
};

123. 买卖股票的最佳时机 III#

本题目具体限制了可以购买两次,因此定义四个状态buy1, buy2, sell1, sell2

  1. 动态规划1

在第一种方法中我们这么定义上面的四种状态:

  • buy1: 第一次购买股票后当前剩余的钱
  • sell1: 第一次卖出股票后当前剩余的钱
  • buy2: 第二次购买股票后当前剩余的钱
  • sell2: 第二次卖出股票后当前剩余的钱

与121. 买卖股票的最佳时机类似,对于Leetcode.121,按照当前的状态我们可以这么写

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int buy = INT_MIN, sell = 0;
        for (int e : prices) {
            buy = max(buy, -e);
            sell = max(sell, e + buy);
        }
        return sell;
    }
};

比葫芦画瓢,我们可以这么写本题的solution.

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int buy1 = INT_MIN, buy2 = INT_MIN;
        int sell1 = 0, sell2 = 0;
        for (int e : prices) {
            buy1 = max(buy1, -e);
            sell1 = max(sell1, buy1 + e);
            buy2 = max(buy2, sell1 - e);
            sell2 = max(sell2, buy2 + e);
        }
        return sell2;
    }
};
  • 时间复杂度: $O(n)$
  • 空间复杂度: $O(1)$
  1. 动态规划2 我们也可以这么解释这四种状态:
  • buy1: 第一次购买的股票价格净值
  • sell1: 第一次卖出股票后剩余的钱
  • buy2: 第二次购买股票的价格的净值
  • sell2: 第二次卖出股票后剩余的钱

同样对于121. 买卖股票的最佳时机我们可以这样写:

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int buy = INT_MAX, sell = 0;
        for (int e : prices) {
            buy = min(buy, e);
            sell = max(sell, e - buy);
        }
        return sell;
    }
};

按照这种理解思路,本体的solution如下:

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int buy1 = INT_MAX, buy2 = INT_MAX;
        int sell1 = 0, sell2 = 0;
        for (int e : prices) {
            buy1 = min(buy1, e);
            sell1 = max(sell1, e - buy1);
            buy2 = min(buy2, e - sell1);
            sell2 = max(sell2, e - buy2);
        }
        return sell2;
    }
};
  • 时间复杂度: $O(n)$
  • 空间复杂度: $O(1)$

188. 买卖股票的最佳时机 IV#

本题只不过将第一题的一次买卖机会,第三题的两次买卖机会变成了k次机会,更加通用,但其中动态规划的思路是一样的,不再多解释.

class Solution {
public:
    int maxProfit(int k, vector<int>& prices) {
        if (prices.size() == 0 || k == 0) return 0;

        vector<int> dp(2 * k + 2, 0);

        for (int i = 1; i <= k; i++) {
            dp[2 * i] = INT_MIN;
            dp[2 * i + 1] = 0;
        }

        for (int e : prices) {
            for (int i = 1; i <= k; i++) {
                dp[2 * i] = max(dp[2*i], dp[2 * i - 1] - e);
                dp[2 * i + 1] = max(dp[2 * i + 1], dp[2 * i] + e);
            }
        }
        return dp[2 * k + 1];
    }
};
  • 时间复杂度: $O(n * k)$
  • 空间复杂度: $O(k)$

309. 最佳买卖股票时机含冷冻期#

需要注意,这里增加了冷冻状态,因此状态方程也发生了变化.

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int n = prices.size();
        vector<int> buy(n), sell(n), cold(n);
        buy[0] = -prices[0];
        for (int i = 1; i < n; i++) {
            sell[i] = buy[i-1] + prices[i];
            buy[i] = max(buy[i-1], cold[i-1] - prices[i]);
            cold[i] = max(cold[i-1], sell[i-1]);
        }
        return max(sell[n-1], cold[n-1]);
    }
};
  • 时间复杂度: $O(n)$
  • 空间复杂度: $O(n)$ 可以优化到$O(1)$

714. 买卖股票的最佳时机含手续费#

与122. 买卖股票的最佳时机 II,基本相同只不过是增加了卖出股票的费用. 转移方程为:

$$ \left{ \begin{array}{l} buy = max(buy, sell - prices[i]) \
sell = max(sell, buy + prices[i] - fee) \
\end{array} \right. $$

class Solution {
public:
    int maxProfit(vector<int>& prices, int fee) {
        int buy = INT_MIN, sell = 0;
        for (int e : prices) {
            buy = max(buy, sell - e);
            sell = max(sell, buy + e - fee);
        }
        return sell;
    }
};
